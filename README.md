# Let Me Tell You a Story — 25 Letters

A bespoke static website presenting *Let Me Tell You a Story*, a collection of 25 letters by Ali Al Mokdad.

## About

This is a lightweight, privacy-first static site built with HTML, CSS, and a little JavaScript. It works offline via a service worker and is designed for a calm, distraction-free reading experience.

## Structure

- `index/` — landing page and entry point
- - `letters/` — the 25 letters
  - - `about/` — background and author information
    - - `assets/` — styles, scripts, and media
      - - `sw.js` — service worker for offline reading
